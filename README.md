I'm **Raghu Sattu** — full-stack developer, security engineer, and someone who prefers understanding systems by taking them apart.

I work across application security, email security, endpoint detection, and automation.  
Most of my time goes into either testing systems in real environments or building tools to remove manual work from those processes.

I don’t follow fixed roles. If something needs to be built, I build it. If something needs to be tested, I test it.
